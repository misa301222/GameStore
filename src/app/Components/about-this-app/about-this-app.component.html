Info/Features

This page was created for educational/learning purposes. It's an Online Store Simulation. Here you have multiple users with multiple roles. You can practically add or delete every product.


JWT Authentication

JWT Authentication handled by Back-End.



Entity Framework

Handled by Back-End. Highly efficient and fun to use.



Users with Different Multiple Roles

You can create user with User or Admin role. Obviously, the Admin can control all the important data in the App. There's a Funds column. It's the money they can spend in the store.



Buy Product Component
Product Section

This is one of the main components. Here you can add reviews. Like or dislike reviews. Delete the own user review and Add product to Cart.



Review Section

Here you can like or dislike reviews. If you left a review in a previously purchased product. The verified text will appear. Otherwise, the not verified text will be displayed.
